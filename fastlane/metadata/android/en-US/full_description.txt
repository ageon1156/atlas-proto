Atlas is a tool for using Android devices with open-source, off-grid mesh radios. This app allows you to manage your mesh devices and communicate with other users.

**Features**

- Mesh networking support
- Off-grid communication
- Radio device management
- Secure messaging

**Community and Support**

This project is currently in beta. If you have questions, feedback, or encounter any problems, please report them through the GitHub issues tracker.

**Documentation**

For more information about features and capabilities, please refer to the project documentation.
